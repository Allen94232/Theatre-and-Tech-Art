# Theatre and Tech Art

Unity project for theatre and tech art experiments and production work.

## Project Info

- Unity version: 6000.0.23f1
- Main scene: `Assets/Scenes/Paint 3D.unity`
- Package source: Unity Package Manager with the Keijiro scoped registry for `jp.keijiro`

## Included Packages

- URP
- Input System
- Timeline
- Recorder
- Visual Scripting
- 2D Feature Set
- OSC / NDI related packages

## Getting Started

1. Open the project in Unity Hub using Unity `6000.0.23f1`.
2. Let Unity finish importing packages and generating project files.
3. Open the main scene from `Assets/Scenes/Paint 3D.unity`.
4. Run the project from the Editor.

## Repository Notes

- This repository is configured to ignore Unity-generated folders such as `Library`, `Temp`, `Obj`, and build outputs.
- Source assets, scenes, packages, and project settings should be committed.
